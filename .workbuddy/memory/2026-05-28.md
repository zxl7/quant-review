# 2026-05-28 工作记录

## 修复：板块共振 pcp 取到个股涨跌幅的 bug

**问题**：盘中盯盘页"板块共振"显示的涨幅全是 -9.95%、-9.9% 等跌停值，明显不对。

**根因**（`web/src/composables/useIntradayAlertPool.ts` `detectResonance`）：
1. 取 `pcp` 时搜索范围是 `freshItems`（仅本轮新拉数据），可能找不到板块异动本身
2. `find` 匹配到的是个股异动（个股的 `relatedNames` 包含该板块），取到的是**个股涨跌幅**（如跌停 -9.95%），而非板块涨跌幅

**修复**：
- 共振计数改为基于 `items.value`（全局积累的全部 items），避免漏判
- 取 `pcp` 时优先匹配 `isPlate && title === sector` 的板块异动 item，没有才 fallback 到个股

**文件**：`web/src/composables/useIntradayAlertPool.ts`

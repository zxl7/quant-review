## AI 分析模块 — 数据加工层 Layer 2

创建了 `daily_review/ai/` 目录，实现三层分离架构：
- **Layer 1**（数据源）：biying API → market_data JSON（不变）
- **Layer 2**（数据加工）：`daily_review/ai/analyzer.py` + `_inject_ai_analysis()` in cli.py
- **Layer 3**（渲染）：Vue3 web 项目

AI 接管字段：
- `summary3.lines` → PlanPage（预测页）
- `learningNotes.tips/quotes` → ShortReminderFooter（底部短线提醒）
- `actionAdvisor.summary` → SentimentPage（操作建议）

工作流：
1. Python pipeline 生成 market_data JSON（不变）
2. AI 预生成分析 → `cache/ai_analysis-YYYYMMDD.json`
3. `cli.py::_inject_ai_analysis()` 在 pipeline 后、HTML 渲染前读取并注入
4. 若 AI cache 不存在，静默回退到 Python 模板文本

Vue3 消费字段（已验证兼容）：
- `summary3.lines` in PlanPage.vue
- `learningNotes.tips/quotes` in ShortReminderFooter.vue
- `actionAdvisor.summary` in SentimentPage.vue
